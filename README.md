# Marketing-and-lead-performance-Dashboard
This project showcases business analytics techniques using Excel, Power BI, and Python (Jupyter Notebook) to perform data analysis, generate insights, and build interactive dashboards for decision-making.

# Excel sheet
Excel dataset used for data exploration, data cleaning, and preliminary analysis.
Includes pivot tables, slicers, descriptive statistics, and raw-to-processed transformation of data.
- Excel
Data cleanup, pivot analysis, basic dashboards

# Python
Jupyter Notebook implementing:
1. Data exploration using Python (Pandas, NumPy).
2. Descriptive statistics and data quality checks.
3. Exploratory Data Analysis (EDA).
4. Charting and trend analysis using Matplotlib/Plotly.
5. Insight generation and observation summary.
- Python (Jupyter)
Data analysis, EDA, statistical insights

# Power Bi
Power BI dashboard designed to visualize business KPIs and uncover insights such as:
1. Sales trends and revenue distribution.
2. Category and region-wise performance analysis.
3. Profitability insights and customer behavior.
4. Dynamic filtering using slicers and drill-through reports

- Power BI
KPI dashboards, DAX-based insights, visual storytelling

# Key Business Questions Addressed

1. Which products generate the highest revenue and profit?
2. What are the month-wise and region-wise sales trends?
3. Which customer segments contribute most to business growth?
4. Where are there opportunities to improve performance?
